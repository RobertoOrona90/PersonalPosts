CRUD para las migraciones

Crear nueva migracion

Add-Migration customer -context ClientesDbContext

Actualizar la base de datos

update-database -migration customer -context ClientesDbContext

Eliminar ultima migracion sin afectar base de datos

Remove-Migration -context ClientesDbContext

Eliminar ultima migracion sin afectar base de datos

Remove-Migration -Force -context ClientesDbContext
